# Página dos Personagens dos X-Men

Este é um repositório que contém uma página interativa com os personagens dos X-Men. O projeto utiliza HTML, CSS, JavaScript para a animação do seletor de personagens e também aproveita o framework Bootstrap para criar um design responsivo.

## Pré-visualização

![X-Men](https://github.com/Rodrigo-Balestrin/X-MEN/assets/126715583/975fd6ad-ebad-4ea7-9142-b0852bae9023)


## Características

- Design moderno e estilizado utilizando Bootstrap.
- Lista de personagens dos X-Men com imagens e descrições.
- Seletor interativo de personagens com animações em Javascript.
- Layout responsivo que se adapta a dispositivos móveis, tablets e desktops.
